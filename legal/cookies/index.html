Cookie Notice

Short version

maxineplatform.com does not set tracking, analytics, or advertising cookies. No consent banner is shown because no consent is required.

What this site uses

  • No third-party cookies. We do not load Google Analytics, Meta Pixel, or any advertising/tracking pixels.
  • Strictly necessary session storage may be used by the browser for basic site function (e.g. preserving preferences you enable). This is not shared with third parties.
  • Server access logs (IP, user-agent, path, timestamp) are kept up to 30 days for operational security. They are not used to build personal profiles.

Products and their own disclosures

Changes

If we ever introduce optional analytics (e.g. self-hosted Plausible), this notice will be updated first and a minimal, consent-based approach will be used.
